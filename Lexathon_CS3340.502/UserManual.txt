***Lexathon User Manual***

-Open Mars.exe v4.5 within the zipfile or the program won't read the files.
-In Mars.exe open REKT 2.0.asm file
-Before assembling, go to settings and make sure "Initialize program counter to global 'main' if defined" is enabled
-Assemble and run the file, if you have any issues PLEASE email us at rks150230@utdallas.edu
-Rules option ingame has more info
-10 Points are rewarded per correct word
-Timer runs in the background, 20 seconds are added per correct word
